# 변경 기록

## 0.6.0 - 2026-07-16

### 추가

- 형성평가 양식과 구조화된 명세를 검증하는 `preview_assessment` 도구
- 검토한 계획으로 학생용·교사용·정답지 HWPX를 생성하는 `apply_assessment` 도구
- 문항별 변경 전·후 미리보기와 서술형 채점 기준 검증
- 실제 한글 프로그램용 실기기 인수 테스트 시나리오와 재현 가능한 예제 명세

### 안전성과 신뢰성

- 등록된 정확한 출력 경로에만 세 HWPX와 manifest를 원자적으로 게시
- 학생용 결과의 정답·해설·교사용 정보 누출 차단
- 원본·프로필·계획 변경 감지와 중복 적용 방지
- 실패 시 부분 산출물과 staging 잔여물을 남기지 않는 정리·복구 절차
- 공개 산출물의 로컬 경로, 임시 실행 파일, 런타임 잔여물 위생 검사

### 검증

- 형성평가 회귀 테스트 155개 통과, 15개 환경 의존 테스트 건너뜀
- 형성평가 소스 19개 strict 타입 검사 통과
- 전체 Python 정적 검사와 산출물 위생 검사 통과
